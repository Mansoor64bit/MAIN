# Portfolio Analysis Project

## Overview

This project was completed as part of the Security Analysis and Portfolio Management course.

The objective was to construct and compare multiple investment portfolios using historical stock market data and portfolio optimization techniques.

A diversified portfolio of ₹8,00,000 was allocated across five industries:

- Telecom
- Oil & Gas
- Consumer Electronics
- Metals
- Construction

The project evaluates portfolio performance using return, risk, and risk-adjusted return measures.

---

## Objectives

- Construct diversified investment portfolios
- Compare portfolio allocation strategies
- Analyze industry-wise performance
- Evaluate risk-return trade-offs
- Apply Modern Portfolio Theory concepts

---

## Portfolio Models

### Equal Weight Portfolio

All selected stocks were assigned equal weights.

### Maximum Sharpe Portfolio

Weights were optimized to maximize risk-adjusted returns.

### Maximum Return Portfolio

Weights were allocated to maximize expected return.
